# Gemini Super-Linker 💎

Google Gemini のチャット履歴を自動で整理・保存し、さらに快適に使うための Chrome拡張機能です。
「あのチャットどこだっけ？」という悩みから解放されます。

## ✨ 主な機能

1.  **履歴の自動保存 (Auto-Linker)**
    * Geminiを使っているだけで、閲覧したチャットの「タイトル」と「URL」を自動的にリスト化して保存します。
    * サイドバーから消えてしまった過去のチャットも、一度開けば永久に記録されます。
2.  **サクサク検索**
    * 拡張機能のポップアップから、キーワード入力で過去のチャットを瞬時に検索できます。
3.  **Markdown ダウンロード**
    * ボタンひとつで、チャットの内容を Markdown形式 (`.md`) でPCに保存できます。
    * NotionやObsidianなどのメモアプリへの貼り付けに最適です。
4.  **Gemアイコンのカスタマイズ**
    * 自分だけのGem (カスタムGemini) のアイコンを、好きな画像にドラッグ＆ドロップで変更できます。
5.  **データバックアップ**
    * 保存された履歴リストを JSONファイルとしてエクスポートできます。

## 📥 インストール方法

この拡張機能はChromeウェブストアには公開していないため、以下の手順で導入してください。

1.  このページの緑色のボタン **[Code]** -> **[Download ZIP]** をクリックして、ファイルをダウンロードします。
2.  ダウンロードしたZIPファイルを解凍（展開）します。
3.  Google Chrome を開き、アドレスバーに `chrome://extensions/` と入力して開きます。
4.  画面右上の **「デベロッパーモード」** をオンにします。
5.  左上に出る **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
6.  解凍したフォルダ（`manifest.json` が入っているフォルダ）を選択します。

これでインストール完了です！Geminiのページ (`https://gemini.google.com/`) を開いて使ってみてください。

## 📖 使い方

### 基本操作
* 拡張機能のアイコンをクリックすると、メニューが開きます。
* **検索:** 下部のリストから履歴を探せます。検索窓で絞り込みも可能です。
* **Markdown保存:** 「📝 Markdownで保存」ボタンを押すと、現在開いているチャットをダウンロードします。

### Gemアイコンの変更方法
1.  アイコンを変えたい Gem のチャット画面を開きます（URLに `/gem/` が含まれるページ）。
2.  拡張機能アイコンをクリックします。
3.  **「Gemアイコン設定」** というエリアが表示されます。
4.  点線で囲まれたエリアに、好きな画像をドラッグ＆ドロップ（またはクリックして選択）します。
5.  「適用する」ボタンを押すと、画面上のアイコンが変更されます。

## ⚠️ 注意事項
* この拡張機能は非公式です。Google Geminiの仕様変更により、一時的に動かなくなる可能性があります。
* チャットの本文データ（テキスト）は、Markdownダウンロード時以外は保存されません（プライバシー重視のため、リンクとタイトルのみ自動保存します）。
