# Gemini Super-Linker 💎 v8.0

**Google Gemini のチャット履歴を「自動保存・整理・活用」する、最強のChrome拡張機能。**

「あのチャット、どこだっけ？」「サイドバーから消えてしまった…」
そんな悩みとはもうサヨナラです。Gemini Super-Linker があなたの執事となり、すべての会話を大切に保管します。

## ✨ パワーアップした機能

### 1. 📂 履歴の完全管理 (Auto-Linker & Editor)
* **自動保存:** ページを開くだけで、チャットのタイトルとURLを自動記録。
* **リネーム機能 (New!):** ✏️ボタンでタイトルを自由に変更可能。変更したタイトルはロックされ、自動保存で上書きされません。
* **削除機能 (New!):** 🗑️ボタンで不要になった履歴をリストから削除できます。

### 2. 🔍 爆速インクリメンタル検索
* ポップアップの検索窓に文字を入れるそばから、過去のチャットを瞬時に絞り込みます。
* 膨大なログの中からでも、目的の会話一発で見つけ出します。

### 3. 📝 Markdown ダウンロード
* **「📝 Markdownで保存」** ボタンひとつで、チャット内容を `.md` 形式でダウンロード。
* 画像が含まれている場合の注釈追加にも対応。
* Notion, Obsidian, GitHub などへの貼り付けに最適です。

### 4. 🎨 Gemアイコンのカスタマイズ (D&D対応)
* あなたの作成した **Gems (カスタムGemini)** のアイコンを、好きな画像に変更できます。
* 設定はドラッグ＆ドロップで一瞬。ブラウザを開くたびに、推しのアイコンがお出迎えします。

### 5. 💾 データバックアップ
* 蓄積されたチャットリストを JSONファイルとして一括エクスポート。
* 大切なデータをPCにバックアップできるので安心です。

## 📥 インストール方法

この拡張機能は野良アプリ（非公式）です。以下の手順でインストールしてください。

1.  このリポジトリの緑色のボタン **[Code]** -> **[Download ZIP]** をクリックし、ダウンロード＆解凍します。
2.  Chromeのアドレスバーに `chrome://extensions/` と入力します。
3.  右上の **「デベロッパーモード」** をONにします。
4.  左上の **「パッケージ化されていない拡張機能を読み込む」** をクリックします。
5.  解凍したフォルダ（`manifest.json` が入っているフォルダ）を選択します。

🎉 **インストール完了！** Geminiのページ (`https://gemini.google.com/`) をリロードして使い始めてください。

## 📖 使い方ガイド

### 基本的な使い方
ブラウザ右上の拡張機能アイコン（💎）をクリックすると、メニューが開きます。

* **履歴を開く:** リストのタイトルをクリックすると、そのチャットへ飛びます。
* **タイトル編集:** リスト右側の「✏️」ボタンをクリック。
* **削除:** リスト右側の「🗑️」ボタンをクリック。
* **バックアップ:** 右上の「💾」ボタンでリストをJSON保存。

### Markdown保存
1.  保存したいチャット画面を開きます。
2.  拡張機能メニューを開き、**「📝 Markdownで保存」** をクリックします。
3.  会話内容が `.md` ファイルとしてダウンロードされます。

### アイコンの変更 (Gems専用)
1.  アイコンを変えたい Gems のページ（URLに `/gem/` があるページ）を開きます。
2.  拡張機能メニューを開くと、**「Gemアイコン設定」** エリアが出現します。
3.  点線エリアに好きな画像を **ドラッグ＆ドロップ** します。
4.  「適用」ボタンを押すと、即座にアイコンが反映されます。

## ⚠️ 注意事項 & プライバシー

* **データはローカル保存:** チャット履歴やアイコン設定は、すべてあなたのブラウザ内 (`Chrome Storage Local`) にのみ保存されます。外部サーバーに送信されることはありません。
* **非公式:** Google公式の拡張機能ではありません。Geminiの仕様変更により、機能が制限される場合があります。
* **「Context Invalidated」エラー:** 拡張機能を更新した直後に出ることがあります。Geminiのページをリロード（F5）すれば直ります。
